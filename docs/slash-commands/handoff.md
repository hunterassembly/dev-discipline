# /handoff

Goal: leave a short, high-signal status snapshot so the next person can continue immediately.

## Include

1. Current objective and what is already done.
2. Exact files changed or planned.
3. Validation status (what passed, what is still unverified).
4. Open risks, blockers, and assumptions.
5. Clear next 1-3 actions.

## Output Template

```md
## Handoff

### Objective
...

### Completed
- ...

### In Progress
- ...

### Validation
- Ran:
- Pending:

### Risks / Blockers
- ...

### Next Steps
1. ...
2. ...
3. ...
```
