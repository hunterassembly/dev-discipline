# Active Plans

Keep one markdown plan per active initiative.

Recommended conventions:

- one file per initiative
- explicit status (`pending`, `in_progress`, `blocked`, `done`)
- short next actions (1-3 items)
- link to related docs/issues/PRs

Move completed plans to `docs/plans/completed/`.
